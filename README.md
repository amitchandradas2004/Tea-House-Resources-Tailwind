# 🍵 Tea House - Landing Page

A modern and responsive Tea Shop landing page built using HTML and Tailwind-CSS.  
This mini project focuses on clean layout structure, product showcasing, testimonials, and a newsletter section.

---

## 🌐 [Live Demo](https://amitchandradas2004.github.io/Tea-House-Resources-Tailwind/)

---

## 📌 Project Overview

Tea House is a simple yet elegant landing page designed for a tea brand.  
The website presents featured products, highlights product quality, displays client feedback, and shares news updates.

The design is fully responsive and works smoothly on desktop and mobile devices.

---

## ✨ Features

- Responsive layout
- Hero section with call-to-action
- Featured product cards
- Brand highlight section
- Client testimonials
- News & Events blog preview
- Newsletter subscription field
- Organized footer with useful links

---

## 📑 Website Sections Explanation

### 1️⃣ Hero Section (Banner Area)

The Hero Section is the first part visitors see when they open the website.  
It introduces the Tea House brand and creates a strong first impression.

**Includes:**
- Large headline
- Short description
- “Explore More” button
- Tea product images
- 5.00 rating badge

**Purpose:**  
To attract attention and encourage users to explore the website.

---

### 2️⃣ Our Featured Products

This section displays the main tea products available in the shop.

**Products:**
- Milk Tea  
- Black Tea  
- Lemon Tea  
- Green Tea  

Each product card includes:
- Image
- Product name
- Short description

**Purpose:**  
To showcase the popular tea items in a clean and organized layout.

---

### 3️⃣ Great Tea, Freshly Presented

This section explains the brand story and emphasizes quality.

**Includes:**
- Brand description text
- “Unique Taste” explanation
- “Premium Quality” information
- Supporting images

**Purpose:**  
To build trust and highlight what makes the brand special and different from others.

---

### 4️⃣ Meet Our Super Clients (Testimonials)

This section presents customer feedback.

**Includes:**
- Client message
- Client name
- Client profession

**Purpose:**  
To increase credibility and build confidence about the product quality.

---

### 5️⃣ News & Events

This section works like a blog preview area.

Each card contains:
- Image
- Date
- Title
- Short description
- “Learn More” link

**Purpose:**  
To share updates, promotions, and keep the website dynamic and up-to-date.

---

### 6️⃣ Footer Section

The footer contains navigation and support links.

**Includes:**
- Quick Links
- Services
- Help section
- Newsletter subscription input
- Social media icons
- Copyright text

**Purpose:**  
To provide easy navigation and additional support for visitors.

---
